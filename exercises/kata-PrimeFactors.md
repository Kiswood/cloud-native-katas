# Prime Factors Kata

## Requirements

Write a function called `primeFactors`

- The function takes an integer argument and returns a list of integers. That list contains the prime factors in numerical sequence.

## Example outputs

- 1: []
- 2: [2]
- 3: [3]
- 4: [2, 2]
- 5: [5]
- 6: [2, 3]
- 7: [7]
- 8: [2, 2, 2]
- 9: [3, 3]
- 10: [10, 5]
- 16: [2, 2, 2, 2]
- 20: [2, 2, 5]
