== Requirements

Write a method called "palindrome" that checks if a given string is palindrome or not.

== Tests

- 'a' is a palindrome
- 'mom' is a palindrome
- 'dude' is not a palindrome
- 'mom mom' is a palindrome
- 'mom dad' is not a palindrome
- '      ' is a palindrome?
- '' throws an error?
- 'moM' is a palindrome?
